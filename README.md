# memobird-nw-app
使用 NW.js 构建的简易的 [Memobird](https://www.memobird.shop/) 桌面程序。
A simple [Memobird](https://www.memobird.shop/) desktop app built with NWjs.
- - - - - 
# 构建 & 使用

因为目录包含依赖文件过多，导致体积较大，所以本项目仅提供**项目文件夹**、构建指南和已构建的版本。
请有需求的自行构建即可。
